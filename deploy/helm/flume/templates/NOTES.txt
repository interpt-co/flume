=== Access the web UI ===

Run the following command to forward the HTTP port to your local machine:

  kubectl port-forward -n {{ .Release.Namespace }} svc/{{ include "flume.fullname" . }} 8080:{{ .Values.service.httpPort }}

Then open your browser at: http://localhost:8080

=== Forward logs from Fluent Bit sidecars ===

Configure your Fluent Bit sidecars to send logs to:

  Host: {{ include "flume.fullname" . }}.{{ .Release.Namespace }}.svc.cluster.local
  Port: {{ .Values.service.forwardPort }}

See deploy/fluent-bit-sidecar.yaml in the flume repository for a
complete sidecar configuration example.
